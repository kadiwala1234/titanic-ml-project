## Project Overview
This project uses **supervised machine learning** to predict whether a passenger survived the Titanic disaster.

It is a **binary classification problem**, where:
- `1` → Survived  
- `0` → Did not survive  

The project demonstrates a complete ML pipeline from data preprocessing to deployment.

## Dataset
- Source: Kaggle Titanic Dataset  
- Link: https://www.kaggle.com/c/titanic/data  

### Features include:
- Age  
- Sex  
- Passenger Class (Pclass)  
- Fare  
- Embarked  

## Technologies Used
- Python  
- Google Colab  
- Pandas & NumPy  
- Matplotlib & Seaborn  
- Scikit-learn  
- Joblib  

## Project Workflow

### 1. Data Preprocessing
- Filled missing values (Age → median, Embarked → mode)
- Removed unnecessary columns (Name, Ticket, Cabin)
- Converted categorical variables using one-hot encoding
- Split dataset into training (80%) and testing (20%)

### 2. Exploratory Data Analysis (EDA)
- Survival distribution visualization
- Correlation heatmap
- Key insights:
  - Females had higher survival rates
  - Higher-class passengers survived more

### 3. Model Training
Two models were implemented:
- Logistic Regression  
- Random Forest Classifier  

### 4. Model Evaluation
Metrics used:
- Accuracy  
- Precision  
- Recall  
- F1-score  
- Confusion Matrix  

**Best Model: Random Forest Classifier**

## Results
- Random Forest achieved the highest performance
- Important features:
  - Sex  
  - Fare  
  - Passenger Class
